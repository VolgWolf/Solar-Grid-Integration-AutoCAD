# Solar-Grid-Integration-AutoCAD
## READ ME

Status: WIP

Project Overview:
Implementing grid-tied solar photovoltaic system into traditional residential power grid using AutoCAD-Electrical. The aim for this project is to
both gain profiency using the AutoCAD-Electrical software along with integrating renewable energies into an existing power grid.

Objectives:
  - Design an SLD for a grid-tied solar PV system using AutoCAD-Electrical
  - Develop electrical schematics for system components such as string inverter, combiner box, and grid interconnection
  - Perform load analysis to determine the size of the PV system
  - Follow NEC and NFPA standards
  - Demonstrate renewable energy integration into an existing power grid

Tools/Softwares:
  - AutoCAD-Electrical
  - NEC standards reference
  - NFPA standards reference
  - IEEE standards reference

Deliverable(s):
  -Single Line Diagram (SLD)
  - AC/DC Wiring Schematics 
  - Combiner box layout
  - Panel Schedule
  - Load Analysis Documentation
